## ✨ Overview

**Apex Fintech Dashboard** is a premium, modern financial analytics dashboard built with cutting-edge web technologies. It features real-time financial tracking, interactive charts, AI-powered insights, and a sleek dark UI.

### 🎯 Perfect for:
- Fintech startups
- Personal finance management
- Banking dashboards
- Crypto analytics
- Expense tracking applications

### 🔥 Key Features:
- 📊 **Real-time Analytics** - Interactive charts and statistics
- 💸 **Transaction Tracking** - Detailed transaction logs and history
- 🤖 **AI Insights** - AI-powered financial advisor widgets
- 🌙 **Premium Dark UI** - Modern, sleek dark theme design
- ⚡ **Fast & Responsive** - Built with Vite for lightning-fast performance
- 🎨 **Beautiful Components** - Reusable, well-designed React components

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/syedmushtaq4033/apex-fintech-dashboard.git
   cd apex-fintech-dashboard
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   ```bash
   cp .env.example .env.local
   # Add your GEMINI_API_KEY and other configurations
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

   The app will be available at `http://localhost:3000`

### Build for Production

```bash
npm run build
npm run preview
```
